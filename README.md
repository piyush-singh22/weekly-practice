# weekly-practice
this repo wiil be used to showcase as well store the mini projects and practice programs every weekends

To add an program file in repo steps :-
# Step 1: Go to your project folder
cd path/to/your/project

# Step 2: Initialize Git (only if not already done)
git init

# Step 3: Add your GitHub repository as remote
git remote add origin https://github.com/<your-username>/weekly-projects.git

# Step 4: Stage your file
git add basic.py

# Step 5: Commit your changes
git commit -m "Add basic.py to weekly-projects repo"

# Step 6: Push to GitHub
git push -u origin main
